INSTAGRAM STATIC SIDEBAR ELEMENTS

Goal: Provide a stable list of Instagram sidebar UI components that the extension can detect using predictable selectors (primarily aria-label). These elements have fixed positions in the viewport and do not move on scroll. The extension will extract bounding boxes for these components.

STATIC NAVIGATION ELEMENTS (LEFT SIDEBAR)

Clickable Target Rule

Purpose / Behavior
